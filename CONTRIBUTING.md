# Contributing to Awesome AI Research Writing

Thank you for your interest in contributing! This project is a community effort to democratize access to high-quality research writing techniques.

## How to Contribute

### Adding New Prompts

1. **Test the prompt thoroughly** with multiple large language models to ensure quality output
2. **Follow the existing format**: Include Role, Task, Constraints, and Output sections
3. **Provide context**: Explain when and why this prompt is useful
4. **Include examples**: Show an example use case when possible

### Improving Existing Content

- Fix typos, grammar, or clarity issues
- Suggest improvements to prompt effectiveness
- Share your experience using a particular prompt
- Propose refinements based on real-world testing

### Adding New Skills Documentation

- Document your experience with agent skills in research workflows
- Provide step-by-step tutorials for skill setup
- Share use cases and practical examples
- Contribute to skills configuration guides

### Reporting Issues

- Found a prompt that doesn't work well? Let us know!
- Suggestions for new prompt categories? Open an issue
- Broken links or outdated references? Please report them

## Submission Guidelines

1. **Fork the repository** and create a feature branch
2. **Test your changes** thoroughly
3. **Write clear commit messages** explaining your changes
4. **Submit a pull request** with a description of your contribution

## Content Standards

- All prompts should be tested with current models
- Maintain academic integrity and accuracy
- Avoid overly complex language in prompt descriptions
- Include relevant citations and references
- Ensure examples are realistic and helpful

## Community Guidelines

- Be respectful and constructive
- Acknowledge the original authors/sources
- Share knowledge generously
- Help others improve their research writing

## Questions?

If you have questions about contributing, please open an issue or reach out to the community.

**Thank you for helping make academic research writing better for everyone!**
